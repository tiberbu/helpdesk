🔍 Mobile Debug Console

Look for a floating icon in the bottom right corner - click it to open the console.

How to use:

  1. Click the console icon (bottom right)
  2. Go to the "Console" tab
  3. Click "Go to Helpdesk App" below
  4. Try to create a ticket
  5. Any errors will appear in the console
📱 Go to Helpdesk App

What to look for:

  • Red error messages - Copy the full text
  • File names and line numbers - e.g., "TicketCustomerSidebar.vue:123"
  • Stack trace - Shows the sequence of function calls
